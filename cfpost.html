Advanced Storage Network finally fixes your storage problem. This add-on adds new devices that you can use to build a storage network that can hold infinite items, including all modded items.


The following information is all available in the Advanced Storage Network tutorial book given to players when they spawn. It can also be crafted if you lose it.


Storage Network

  • A network of devices that can hold an infinite amount of items.
  • Comprises one Storage Core and any number of Storage Drives and Storage Interfaces connected by Storage Cables. Can also contain many other optional devices.
  • To get started, you will need one Storage Core, one Storage Drive, and one Storage Interface. Read their respective tutorial entries for more information.

Storage Core

  • The core of a storage network.
  • Required for a storage network to function.
  • Only one of these blocks exists in a network.

Storage Cable

  • Connects the devices in a storage network.
  • All devices in a network need to be connected with these cables.

Storage Drive

  • Provides the network it's connected to with 1,000u of storage space.
  • Drops a Storage Disk (With Data) when destroyed that contains the items the drive had before it was broken.
  • Any number of these devices can exist in a network.

Storage Interface

  • Allows players to add, remove, or view items in a storage network.
  • Interact with the interface while holding any item to put it in the network.
  • Any number of these devices can exist in a network.

Storage Import Bus

  • Imports items from the container it's facing into the storage network it's connected to.
  • Stops importing when receiving a redstone current.
  • Any number of these devices can exist in a network.

Storage Export Bus

  • Exports items from the storage network it's connected to into the container it's facing.
  • More filter options are available when exporting an item that is enchantable or breakable.
  • Stops exporting when receiving a redstone current.
  • Interact with an empty hand to configure.
  • Any number of these devices can exist in a network.

Storage Level Emitter

  • Tests that the total amount of a filtered set of item stacks in the storage network matches a query.
  • Filter options are available when exporting an item that is enchantable or breakable.
  • Activates the repeater in front of it when the test is true.
  • Interact with an empty hand to configure.
  • Any number of these devices can exist in a network.

Storage Disk

  • A crafting item.
  • Can contain data (will say "Storage Disk (With Data)").
  • Interact with an empty Storage Drive while holding a disk with data to deposit the items inside of the disk into the drive.

Storage Crystal

  • A crafting item.
  • Created by smelting a Raw Storage Crystal (a craftable item, see recipe book) in a furnace or blast furnace.

Add-on Configuration

  • Use the /scriptevent fluffyalien_asn:addonrule <rule> [value] command to change a rule.
  • Use the /scriptevent fluffyalien_asn:addonrule help command for a list of rules.
  • Omit [value] from the command to get the current value of the rule.

Enable all experimental toggles under "Add-on Creators"
